# Documentación de Page Objects — PACS (OHIF + Keycloak + Orthanc)

## 1. LoginPage
Representa el formulario de Keycloak usado para iniciar sesión.

## 2. ViewerPage
Representa la interfaz de OHIF Viewer incluyendo Study List y visor.

## 3. OrthancAdminPage
Página de administración PACS accesible solo para usuarios pacsadmin.
